**<h2 align="center">&#128204; Módulo 3 - Fundamentos de Bases de Datos Relacionales</h2>**

[GitHub Pages - Proyectos Módulo 3 - Bootcamp Desarrollo Aplicaciones Móviles](https://kathyalde21.github.io/ejercicios_bootcamp_app_mov/sitiosModulo3.html)

<hr style="border: none; height: 0.06px; background-color: grey; width: 100%">

<p>
En este módulo trabajé con bases de datos relacionales utilizando MySQL, enfocándome en comprender cómo modelar la información, relacionar tablas y construir estructuras que respondieran a distintos contextos de uso.
</p>

<p>
Gran parte del trabajo se desarrolló de forma progresiva, retomando una misma base de datos en varias etapas para ampliarla, modificarla y consultarla. Eso me permitió ver con más claridad cómo evoluciona un modelo relacional a medida que aparecen nuevos requerimientos.
</p>

<p>
A lo largo del módulo practiqué creación de tablas, relaciones, consultas, inserción de datos y ajustes sobre modelos existentes. Los ejercicios se concentraron especialmente en tres líneas: sistemas de ventas, prevención de riesgos y modelos entidad-relación aplicados a distintos casos.
</p>

<hr style="border: none; height: 0.06px; background-color: grey; width: 100%">
